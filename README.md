# Lap Counter

A ros2 package to count how many laps the car has made.

## Getting Started

### Dependencies

* Ros2 Humble
* [lart_msgs](https://github.com/FSLART/lart_msgs)

### Installing

* clone this repository to {workspace}/src
* build the package

### Executing program

* source ros
* source your workspace
* run this command:
```
ros2 run lap_counter launch
```
